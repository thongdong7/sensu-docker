# About

A docker-compose project to build a Sensu System (Sensu Core, RabbitMQ, Redis, Sensu Dashboard (Uchiwa))

# Get started

Clone this project

Navigate to the docker folder and type

```
cd docker
docker-compose up
```

Open `http://localhost:3000` to access to Sensu Dashboard

# Sample sensu-client

```
cd sample/docker
docker-compose up
```
